PTT Article Parser
==================
一個用來分析 PTT 文章的工具。

Features
--------
* 根據文章標題，重新命名檔案名稱
* 白金緩慢增加中

Dependencies
------------
Install python 3.
https://www.python.org/

Install `docopt`.
```
pip install docopt
```

Usage
-----
```
Usage:
  pap.py rename [--format=<format>] <file>...
  pap.py --help
	
Options:
  -h --help             Show this.
  -f --format=<format>  Set output format. 
                        [default: [@board] @title by @author.@time.ans]
```
Example:
```
pap.py rename "file1" "file2" "file3"
```
